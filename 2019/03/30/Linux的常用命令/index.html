Linux的常用命令

发表于 2019-03-30 | 更新于 2019-03-31 | 评论数：

创建目录：

mkdir 目录名

删除文件或文件夹：

rm 文件名
rm -r 文件夹

查看当前目录下文件

ls
ls -a（全部）
ls -l（细节）

转到目录下

cd 目录路径（例如./sxy/2019/03/31）
cd .（当前目录）
cd ..（上一目录）

文件夹重命名或移动

重命名：mv a（旧的） b（新的名字）
移动：mv a /b/c（把a移到b下面，并重命名为c）

文件或文件夹的拷贝

cp (-r) 当前文件 目标路径
主机与服务器之间的拷贝（在主机上操作）：scp (-r) 当前文件 目标路径
从主机到服务器：scp (-P 端口号) (-r) 主机路径 root@地址:/服务器上的目标路径
从服务器到主机：scp (-r) root@地址:/服务器上的目标路径 主机路径

mac远程连接Linux系统

（可能需要获取接入Linux的权限，在Linux上添加mac的ssh公钥）：
ssh root@地址
ssh -p 端口号 root@地址

Linux下查看配置

nvidia-smi

查看进程

ps
ps -a （全部）
ps -xef （细节）
kill -9 进程号 （kill进程）

解压缩

tar zxvf 文件名.tgz -C 解压到哪个目录
